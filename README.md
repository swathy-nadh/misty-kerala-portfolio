# 🪄 Wizard Portfolio

An immersive fantasy-inspired developer portfolio built using React, Tailwind CSS and Framer Motion.

Inspired by magical libraries, enchanted forests, ancient castles, handwritten spells and mysterious wizarding worlds.

## Features

✨ Animated handwritten name

✨ Floating magical particles

✨ Spellbook project showcase

✨ Potion-inspired skills

✨ Owl Post contact section

✨ Responsive

✨ Smooth scrolling

✨ Dark fantasy aesthetic

## Tech Stack

- React
- Tailwind CSS
- Framer Motion
- Vite
